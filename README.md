# RemoteView — Трансляция экрана через браузер

Аналог TeamViewer/AnyDesk, работающий прямо в браузере без установки. Хост делится экраном по уникальной ссылке, зритель открывает ссылку и видит экран в реальном времени.

## Как работает

- Хост заходит на сайт → нажимает "Начать сессию" → получает уникальную магик-ссылку
- Отправляет ссылку зрителю
- Зритель открывает ссылку → видит экран хоста
- Никто без ссылки подключиться не может (токен в ссылке)
- Видео идёт напрямую P2P (WebRTC), трафик не через сервер

## Технологии

- **Backend:** Node.js + Express + WebSocket (ws) — только сигнальный сервер
- **Frontend:** Vanilla HTML/CSS/JS
- **Видео:** WebRTC + `getDisplayMedia()`
- **Безопасность:** уникальный токен (48 hex символов) в каждой ссылке

## Структура файлов

```
remote-view/
├── server.js          # Сигнальный сервер (WebSocket + REST)
├── package.json
└── public/
    ├── index.html     # Страница хоста
    ├── join.html      # Страница зрителя
    └── style.css      # Тёмный UI
```

## Запуск локально

```bash
npm install
npm start
# Открыть: http://localhost:3000
```
